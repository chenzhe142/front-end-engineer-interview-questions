- `img`: it acts like `inline-block` elements in some browser
- `input`
- `a`
- `em`
- `strong`
- `button`
- `label`
- `select`
- `textarea`

### inline elements only have wrapped height & width. It means it will do nothing if you are setting `width` or `height` for it.
